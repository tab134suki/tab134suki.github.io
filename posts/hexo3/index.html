#vercel部署和绑定自己的域名！
这一步很长，耐心看完（写完这个这篇文章就不写了，太长了）
1.首先，在 Vercel 官网：https://vercel.com/ 注册一个新账户（注册新用户必须使用 Github、Gitlab 或者 Bitbucket 的账户进行授权，并绑定手机号。）
注册完成后，可以在配置页面修改自己的邮箱地址。
这里建议使用 Github 进行授权登陆，后续可以选择 Github 上的项目直接部署也会很方便的。
2.登陆系统后点击右上角的Add New…再点击project，创建新项目
在“Import Git Repository”中选择你创建的项目。再点击“import”
3.点击下方的Deploy就会自己部署了。
但因为vercel被墙了，所以大陆是无法访问的。
但是不要慌，我们可以绑定一个自己的域名。
4.先购买一个域名。（看自己的需求和实际财富进行购买）（这里以阿里云为例）
进入https://wanwang.aliyun.com/ 输入你想要注册的域名名称和一级域名
然后按照要求购买…（具体怎么办可以看阿里云的文档）
5.买完后返回vercel，点击你刚刚部署的页面
6.点击上面那一行的“settings-Domains”输入自己的域名，并在自己域名解析设置里添加
记录类型：CNAME，主机记录写www，记录值：cname-china.vercel-dns.com
提示：可以在记录值直接填写自己原来的域名
这样就可以了
